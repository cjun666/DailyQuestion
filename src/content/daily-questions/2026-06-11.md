类型与中括号紧挨相连来表示数组,中括号是类型的一部分,而不是变量的一部分。正例:int[] arrayDemo。


**选项分析:**

- **选项 A 正确**: String[] args
- **选项 B**: String []args
- **选项 C**: String [ ] args
- **选项 D**: String args[]
